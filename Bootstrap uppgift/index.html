Begrepp

HTML-element

Har oftast en starttagg, innehåll och en sluttagg. Vissa element är dock tomma och saknar då innehåll och sluttagg.

HTML

HTML styr front-end delen av hemsidor. Är en del av den grundläggande standarden för WWW.

ID

Ger element ett id attribut med ett unikt värde. Likt class men kan bara användas av ett element.

Class

Ger en class till element. Kan till skillnad från id användas av flera element.


Meta

Metadata är data som beskriver data. Meta elementet är ett sätt att lägga till metadata till ett dokument.

Responsive

Ett begrepp som används inom webbdesign som beskriver om en websida anpassar sig till skärmens storlek och upplösning.

HTTP

En annan del av den grundläggande standarden för WWW. Det kommunikationsprotokoll som används för att överföra HTML dokument.

HTTPS

Ett protokoll för överföring av krypterad information mellan webbserver och webbläsare.


DNS

En databas som bland annat kopplar domännamn med IP-adresser.

CDN

En viktig del i hur innehåll som ljud, bilder, och videor levereras på webben. Avlastar servern vilket leder till högre prestanda och mindre fördröjning.

Compressed data

Datan omkodas så att antalet bits som krävs för att repressentera datan minskar.

Local bootstrap

När man använder Bootstrap via ett CDN kan problem uppstå på sidan när man inte är ansluten till något nätverk, att ladda ner Bootstrap och köra det lokalt fixar detta problem.


JQuery

Ett JavaScript bibliotek som ska göra det enklare att använda JavaScript.

Flexbox

Ska göra det enklare och mer effektivt att placera och ändra utrymme mellan objekt i en container. Ska göra sidor mer responsive.

Grid

Delar in sidan i ett rutnät som gör webbdesign enklare, likt Flexbox fast med lite olika användningsområden.

Code snippets

En bit kod som gör så att programmeraren inte måste skriva repitetiv kod.


WASM

En typ av kod som kan köras på morderna webbläsare. Kan användas tillsammans med JavaScript.

Chromium

En open-source webbläsare som flera andra webbläsare är baserade på.

SVG

Ett format för tvådimentionell vektor grafik. Baserat på XML.

PNG24

PNG är ett format för digitala bilder. PNG24 stöttar 24 bits av färger (över 16 miljoner).
